CS291A Intro Project (project 0)

This project will teach us/refresh our knowledge of basic HTML, CSS, and Git

There are a few requirements for this project including

  • A title and heading
  • An image hosted in the GitHub repo
  • A CSS file hosted in the GitHub repo
  • An unordered list with at minimum 3 elements
  • A table with 2 columns and 3 rows
  • A hyperlink to the GitHub repo

Now for a some pros and cons of using bootstrap in your website/webapp (we are not allowed to in this project)

Pros Cons
A consistent framework to use Can be limited -- may need to override some parts of the bootstrap
Makes development faster (ie use a bootstrap package/framework instead of developing your own) Documentation can be poor sometimes
Can help you avoid errors/issues (already solved by the bootstrap) Bootstrap can have errors/unexpected behavior
